## 🌐 URLs
- **Produção:** https://www.balneariohair.com.br
- **Cloudflare Pages:** https://balneario-hair-institute.pages.dev

## 📁 Estrutura
```
/                     → Home institucional
/transplante-capilar/ → Página de técnicas
/transplante-capilar/fue/ → FUE Sapphire
/transplante-capilar/dhi/ → DHI Choi
/sobre/               → A Clínica
/resultados/          → Galeria de Resultados
/depoimentos/         → Depoimentos

```
